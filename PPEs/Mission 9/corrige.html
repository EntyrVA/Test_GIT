Corrigé des Questions

1. Quel protocole est utilisé pour envoyer des emails ?

Réponse correcte : SMTP

2. Quelle est la fonction principale d'un serveur DNS ?

Réponse correcte : Résoudre les noms de domaine en adresses IP

3. Quel langage est principalement utilisé pour structurer le contenu d'une page web ?

Réponse correcte : HTML

4. Que permet une clé étrangère dans une base de données relationnelle ?

Réponse correcte : Assurer l'intégrité référentielle entre deux tables

5. Que signifie l'acronyme CSS en développement web ?

Réponse correcte : Cascading Style Sheets

6. Quelle est la principale fonction d'un serveur proxy ?

Réponse correcte : Agir comme intermédiaire entre un client et un serveur

7. Quelle commande Linux permet de lister les fichiers d'un répertoire ?

Réponse correcte : ls

8. Que signifie l'acronyme API en programmation ?

Réponse correcte : Application Programming Interface

9. Quel est le rôle d'un serveur FTP ?

Réponse correcte : Transférer des fichiers entre un client et un serveur

10. Que signifie l'acronyme VPN ?

Réponse correcte : Virtual Private Network
